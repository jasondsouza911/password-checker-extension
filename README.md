CyberGuard is a cyber-themed browser extension that provides **real-time password strength feedback** while users type passwords on websites, including secure platforms like Instagram and enterprise portals.

## Features
- Live password strength indicator (Weak / Moderate / Strong)
- Works on modern websites (React, Angular, secure logins)
- Cyber-themed UI
- Secure password suggestions via extension popup
- Keyboard shortcut support
- Compatible with Chrome, Edge, and Brave

## How it works
- Displays a non-intrusive badge near password fields while typing
- Suggestions and detailed checks are available via the extension popup
- Designed to avoid interfering with secure website behavior
